Field Plot

After the FEMAP conversion is complete, the converted VTK result files can be visualized with the Field Plot tool.

This workflow is intended for viewing the field data generated from the converted .pvd output, typically the file created by the FEMAP converter such as .pyemsi/output.pvd.

See Convert FEMAP Files for the conversion step that produces the VTK output used here.

Open The Field Plot Dialog​

Open the field-plot builder from:

  • File -> Field Plot

The dialog lets you choose a converted field file, configure how the result should be displayed, and open the field view inside pyemsi.

Or click on icon on the main toolbar.

Basic Workflow​

  1. Run the FEMAP conversion so the VTK output files are available.
  2. Open the Field Plot dialog.
  3. Select the converted field file, usually the generated .pvd file.
  4. Click the telescope button beside Field File to inspect the file and load the available arrays.
  5. Enable and configure one or more plot stages such as scalar, contour, or vector display.
  6. If you are using vectors, click the telescope button beside Factor to suggest a glyph scale based on about 10% of the mesh size.
  7. Click Plot to open the field view in pyemsi.

Main Inputs​

The top of the dialog contains the main file selection fields.

FieldDescription
Field FileThe converted VTK file to visualize. In most cases this is the .pvd file generated by the FEMAP converter. The dialog defaults to the last FEMAP converter output when available.
TitleThe title used for the generated field-view tab in pyemsi.

Array Discovery​

The telescope button beside Field File reads the selected field file and inspects the available scalar and vector arrays.

When discovery succeeds, the dialog:

  • refreshes the available field names in the scalar, contour, and vector controls
  • updates vector scaling choices based on the detected arrays

The array-backed controls start empty until this discovery step is run.

This is the recommended first step after selecting the field file.

Plot Stages​

The dialog is organized into collapsible plotting stages. You can enable only the stages you need.

Scalar​

The Scalar section colors the mesh using a scalar field.

It includes:

  • Name: the scalar field to display
  • Mode: whether the values are shown from node data or element data
  • Colormap: the colormap used to color the field

Contour​

The Contour section draws contour lines from a selected scalar field.

It includes:

  • Name: the scalar field used for contour generation
  • Contours: the number of contour levels
  • Color: the contour-line color
  • Line Width: the contour-line thickness

Vector​

The Vector section displays vector quantities as glyphs.

It includes:

  • Name: the vector or vector-related field to visualize
  • Scale: how glyph scaling is determined
  • Glyph Type: the glyph shape, such as arrow, cone, or sphere
  • Factor: the multiplier applied to glyph size, with a telescope button to suggest a value from the discovered mesh and array magnitudes
  • Tolerance: an optional tolerance control for glyph generation
  • Color Mode: how glyph coloring is determined

The vector stage is especially useful for magnetic flux density, current density, and force vectors.

Feature Edges​

The Feature Edges section controls the mesh-edge overlay.

It includes:

  • edge color
  • line width
  • opacity

This helps make geometry boundaries easier to read in the final field view.

Script And Plot Actions​

The bottom row of the dialog provides the main actions.

ButtonDescription
Script...Generates a Python script showing the equivalent Plotter commands for the current dialog configuration.
PlotBuilds the field plot and opens it in pyemsi.
CancelCloses the dialog without creating a plot.

The inline telescope buttons handle array discovery and vector-factor suggestion. Plot and Script... require discovered array selections for any enabled scalar, contour, or vector stage.

Result​

When you click Plot, pyemsi creates a Plotter instance, applies the selected stages, and opens the resulting field view inside the GUI.

This allows you to inspect converted FEMAP results interactively after each conversion run.

For the underlying plotting API, see Plotter.
